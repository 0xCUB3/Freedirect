Turn on Freedirect in Safari Extensions to start redirecting.

Freedirect is on and redirecting. Configure services in the extension popup.

Freedirect is turned off. Turn it on in Safari Extensions preferences.

Turn on Freedirect in Safari:

  1. Open Settings → Apps → Safari → Extensions, or search for “Safari Extensions”.
  2. Select Freedirect and turn it on.
  3. Set website access to Allow for all sites.

All redirect settings live inside the Freedirect extension page in Safari.
