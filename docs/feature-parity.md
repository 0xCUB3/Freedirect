## LibRedirect service coverage reference

LibRedirect currently exposes 52 service groups. Freedirect now catalogs all 52 groups in the Safari extension with per-service confidence metadata. Eleven groups have high-confidence bespoke templates; the remaining groups use conservative multi-host path-preserving starter templates and require service-by-service Safari verification before being considered complete parity. See `docs/service-matrix.md` for the generated granular matrix and `docs/service-test-cases.md` for the generated manual runtime checklist.

## Additional user-wanted features backlog

- Guided iOS/macOS permission onboarding with platform-specific instructions; private iOS Settings deep links are intentionally avoided, and iOS/iPadOS 26.2+ state checks are used where available.
- Instance health checks with latency, availability, and stale-result warnings are implemented in extension settings.
- Favorite/pinned instances, custom instance add/remove, selected vs daily-rotating instance mode, public instance snapshots, update-instance-lists, per-service Best instance selection, and safe defaults. Implemented in extension settings; the native app intentionally does not duplicate settings.
- Profiles/presets such as "Strict", "Balanced", and "Manual" plus bulk service enable/disable/reset actions. Balanced now enables YouTube, Reddit, X/Twitter, Instagram, TikTok, and Google Search by default. Implemented in extension settings; more profile tuning pending.
- Native JSON file backup/restore is intentionally deferred; backup/restore lives in the Web Extension settings page.
- SafariDriver-based integration smoke tests for extension loading and DNR behavior.
- Rule preview/debugger explaining generated DNR/session rules. Attributed generated rule preview and URL redirect/reverse debugger are implemented; runtime Safari validation pending.
- Accessibility pass started: extension popup/options now add live-region updates and localized ARIA labels for key controls; deeper VoiceOver/manual keyboard testing remains in Safari.
