Turn on Freedirect in Safari Extensions:

  1. Open Settings.
  2. Go to Apps → Safari → Extensions, or search Settings for “Safari Extensions”.
  3. Select Freedirect and turn it on.
  4. Set website access to Allow for all sites.

All redirect settings live inside the Freedirect extension page in Safari.

You can turn on Freedirect’s extension in Safari Extensions preferences.
